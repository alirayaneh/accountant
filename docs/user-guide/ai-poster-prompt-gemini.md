# Gemini Poster Prompt — Amoza Online Accountant

Copy the prompt below into **Google Gemini** (Image generation / Imagen) or **Gemini Advanced** with image creation enabled.

Optional reference logo: `amoza sas logo.png` from Downloads or brand assets from Amoza.

## Prompt (copy everything inside the block)

```
Create a professional promotional poster for a Persian accounting software product.

## Product
- Name (Persian): حسابدار آنلاین آموزا
- Tagline (Persian): رایگان، حرفه‌ای، برای مغازه‌دار واقعی
- URL: hesab.amoza.ir
- Category: Free accounting + inventory + POS software for small retailers and importers in Iran

## Format & Dimensions
- Vertical poster: 1080 x 1350 pixels (Instagram portrait) OR A4 print ratio
- High resolution, print-ready quality
- RTL (right-to-left) layout — Persian/Farsi text must read naturally right-to-left

## Visual Style
- Modern dark fintech aesthetic matching the app UI
- Background: deep navy / charcoal (#0a0e1a to #1a1f35) with subtle grid or gradient mesh
- Accent colors: electric cyan (#00d4ff) and purple-blue gradient (#6366f1 to #00d4ff)
- Glassmorphism cards with soft glow
- Clean, premium SaaS look — NOT cluttered, NOT cartoonish
- Professional Persian typography (similar to Vazirmatn or modern sans-serif Farsi)

## Layout Structure (top to bottom)

### Header
- Amoza logo placeholder (stylized "آ" or "02" in cyan circle)
- Main headline in large bold Persian:
  **حسابدار آنلاین آموزا**
- Subheadline:
  **حسابداری + انبارداری + فروش — در یک نرم‌افزار**

### Hero Visual (center)
- Mockup of a dark-mode dashboard on a laptop/tablet screen showing:
  - Financial KPI cards (فروش، سود، مخارج)
  - A bar chart with cyan and purple bars
  - RTL sidebar on the right side
- Subtle floating UI elements around the mockup (product card, barcode icon, payment receipt)

### Feature Grid (2x3 or 3x2 icons with Persian labels)
Each icon in a rounded glass card with cyan icon:
1. 📦 **انبارداری** — Barcode & stock management
2. 💱 **قیمت چندارزی** — USD, AED, CNY pricing
3. 🛒 **ثبت فروش سریع** — POS & barcode scan
4. 👥 **مدیریت مشتری** — Customer & debt tracking
5. 📊 **گزارش زنده** — Live dashboard & charts
6. 🤖 **دستیار هوشمند** — AI inventory assistant

### Badge Row (horizontal pills)
Three prominent badges:
- **رایگان** — bright green/cyan pill
- **Linux · Windows · Mac** — platform badge
- **آفلاین + آنلاین** — connectivity badge

### Footer
- Large CTA text: **همین الان رایگان شروع کنید**
- URL: **hesab.amoza.ir**
- QR code placeholder (bottom-left or bottom-right corner) linking to hesab.amoza.ir
- Small text: «نسخه دسکتاپ و وب»

## Mood & Audience
- Target: Iranian shop owners, importers, small business sellers
- Feeling: trustworthy, modern, empowering — "finally a free tool that looks expensive"
- Avoid: stock photo clichés, western currency symbols ($), English-heavy text

## Technical Requirements
- All visible text MUST be in Persian (Farsi)
- RTL alignment throughout
- High contrast for readability
- No watermarks
- No lorem ipsum — use the exact Persian labels provided above

## Do NOT include
- Fake app store ratings
- Misleading "paid" pricing
- Cluttered text blocks
- Low-resolution or blurry elements
```

## Tips for best results in Gemini

1. Upload one app screenshot (`12-dashboard.png`) as **reference image** for color/style matching
2. Upload Amoza logo if available as second reference
3. If text renders poorly in Persian, generate layout without text first, then ask Gemini to add Persian text in a second pass
4. Request **"no English text except hesab.amoza.ir"** if the model adds unwanted English
